# 课程介绍

欢迎来到 **Terraform 交互式教程**！

本教程将带你从零开始掌握 Terraform 基础设施即代码（IaC）的核心技能。你将在真实的云端实验环境中动手实践，无需安装任何软件。

## 🎯 你将学到

- Terraform 的核心概念与工作流（init → plan → apply）
- 使用 HCL 编写基础设施配置
- 管理状态文件与远程后端
- 使用 TFLint 进行代码质量检查
- 使用 conftest 进行合格检查
- 模块化与最佳实践
- **其他的我想到的和 Terraform 有关的内容，我想写什么就写什么**

## ⚡ 实验环境

每个实验章节都嵌入了一个真实的 Linux 终端环境（由 [Killercoda](https://killercoda.com) 提供），预装了：

| 工具 | 用途 |
|------|------|
| Terraform CLI | 基础设施即代码引擎 |
| TFLint | Terraform 代码质量检查 |
| LocalStack | 本地 AWS 服务模拟器（S3、IAM、DynamoDB） |

环境会在你打开页面时自动创建，关闭后 60 分钟自动销毁。

## 📋 前置知识

- 基本的命令行操作（`cd`、`ls`、`cat`）
- 了解 JSON / YAML 格式（非必须但有帮助）
- **不需要** AWS 账号或信用卡

准备好了？点击侧边栏的下一章开始吧！
