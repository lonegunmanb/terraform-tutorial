# 恭喜完成！

你已经掌握了 Terraform 模块的基础知识：

- **根模块** — 执行 `terraform` 命令时的当前目录，是所有配置的入口
- **标准模块结构** — `main.tf` + `variables.tf` + `outputs.tf` 的文件组织方式
- **子模块** — 通过 `module` 块调用，实现代码复用和封装
- **嵌套模块** — 子模块内部继续调用其他模块，形成层级结构
- **作用域隔离** — 模块内外变量和资源互相隔离，通过 `variable` 和 `output` 通信
- **terraform test** — 使用 `.tftest.hcl` 文件编写测试用例，验证模块行为

接下来你可以继续学习模块的创建技巧、远程模块的使用、模块元参数以及模块重构。
