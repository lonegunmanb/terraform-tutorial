# Terraform 模块基础

在这个实验中，你将理解 Terraform 模块的核心概念，并亲手创建和调用模块。

模块是 Terraform 代码组织的基本单元——任何包含 `.tf` 文件的目录就是一个模块。你将通过以下三个步骤掌握模块的基础知识：

1. **根模块与标准结构** — 认识根模块，了解模块的标准文件组织方式
2. **创建并调用子模块** — 查看并运行一个本地子模块示例，体验模块的封装和作用域隔离
3. **测验** — 自己动手编写模块，用 `terraform test` 验证答案

> 💡 本实验使用 LocalStack 模拟 AWS 服务，所有操作都在本地完成，无需真实 AWS 账号。
