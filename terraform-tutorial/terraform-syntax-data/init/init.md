# Terraform 数据源

在这个实验中，你将学习 Terraform 数据源（data）的核心用法——如何查询已有资源的信息，并将其与 resource 结合使用。

数据源是 Terraform 中的**只读**资源：它不会创建或修改任何基础设施对象，而是查询/计算数据供其他地方使用。你将通过以下三个步骤掌握数据源的关键知识点：

1. **数据源基础** — 使用 `data` 块查询 AWS 账号信息和区域信息，并在资源中引用
2. **data + resource 协作** — 先创建资源，再用数据源查询它们，体验数据源的典型使用场景
3. **小测验** — 通过 `terraform test` 检验你对数据源的理解

> 💡 本实验使用 LocalStack 模拟 AWS 服务，所有操作都在本地完成，无需真实 AWS 账号。
