# 恭喜完成！

你已经掌握了 Terraform 模块调用的核心技能：

- **source 来源类型** — 本地路径、Terraform Registry、Git 仓库等
- **version 约束** — 使用精确版本、范围约束或悲观约束锁定模块版本
- **传参** — 通过 module 块参数向模块传入变量值
- **输出引用** — 通过 `module.<名称>.<输出>` 获取模块返回值
- **模块间数据流** — 一个模块的输出可以作为另一个模块的输入
- **count** — 使用数字索引批量创建模块实例
- **for_each** — 使用字符串键批量创建模块实例，增删更稳定
- **depends_on** — 声明模块间的隐式依赖，module 被视为整体单元
- **terraform test** — 使用 `.tftest.hcl` 文件验证模块调用行为

接下来你可以继续学习模块的其他元参数（providers）以及如何使用 moved 块安全地重构模块。
