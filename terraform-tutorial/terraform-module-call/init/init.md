# Terraform 模块调用

在这个实验中，你将学习如何在 Terraform 配置中调用模块——掌握模块的来源指定、版本约束、参数传递和输出引用。

你将通过以下四个步骤掌握模块调用的核心技能：

1. **模块来源与版本约束** — 了解 source 参数支持的各种来源类型，以及如何锁定模块版本
2. **传参与输出引用** — 通过 variable 向模块传参，通过 output 获取模块返回值
3. **使用 count 和 for_each 调用模块** — 批量创建模块实例
4. **测验** — 综合运用所学知识，用 `terraform test` 验证答案

> 💡 本实验使用 LocalStack 模拟 AWS 服务，所有操作都在本地完成，无需真实 AWS 账号。
