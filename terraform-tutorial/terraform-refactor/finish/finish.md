# 恭喜完成！

你已经掌握了 Terraform 代码重构的三种核心工具：

- **import 块** — 将已有基础设施纳入 Terraform 管理，无需销毁重建
- **removed 块** — 从 Terraform 管理中安全移除资源，实际基础设施不受影响
- **moved 块（重命名）** — 给资源或模块起更好的名字，状态自动更新
- **moved 块（提取模块）** — 将资源从根模块提取到子模块，零停机重构

关键要点：

- import 块只能在根模块中使用，但 to 地址可以指向子模块
- removed 和 moved 块可以在任意模块中使用
- moved 块一旦移除是破坏性变更——对公开模块建议永久保留
- 重构前后始终用 terraform plan 确认无意外的销毁操作
- 用 awslocal / aws 命令独立验证实际资源未受影响
