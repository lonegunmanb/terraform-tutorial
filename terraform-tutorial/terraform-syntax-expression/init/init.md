# Terraform 表达式

在这个实验中，你将学习 Terraform 表达式的各种用法。

表达式是 Terraform 配置中用于计算值的核心机制。你将通过四个步骤掌握所有关键知识点：

1. **运算符与条件表达式** — 算术、比较、逻辑运算符和三元条件表达式
2. **字符串模板与函数调用** — 插值、指令（if/for）和内建函数
3. **for 表达式与 splat** — 集合转换、过滤、分组和属性提取
4. **练习与测试** — 动手编写代码，用 `terraform test` 验证你的答案

> 💡 这个实验不需要云服务或 LocalStack，仅使用 Terraform 本地功能（locals 和 outputs）来练习表达式。
