## 核心概念回顾

- **运算符** — 算术（+、-、*、/、%）、比较（==、!=、<、>）、逻辑（&&、||、!）
- **条件表达式** — `condition ? true_val : false_val`，结合 null 实现可选赋值
- **函数调用** — `函数名(参数...)`，展开符 `...` 展开列表入参
- **字符串模板** — `${}` 插值、`%{}` 指令（if/for）、`~` 去除空白
- **for 表达式** — `[for ...]` 输出元组、`{for ... => ...}` 输出对象、`if` 过滤、`...` 分组
- **splat 表达式** — `list[*].attr` 简洁提取列表元素的属性
