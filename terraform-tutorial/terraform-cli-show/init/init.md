# terraform show 实战练习

通过本节三个步骤，你将在真实的 AWS（LocalStack）环境中掌握 `terraform show` 的完整用法。

## 实验环境

LocalStack 已启动，Terraform 已完成初始化并创建了资源（2 个 S3 桶 + 1 个 DynamoDB 表），你可以直接使用 `terraform show` 查看状态。

## 学习内容

| 步骤 | 内容 |
|------|------|
| 步骤 1 | 查看当前状态：人类可读输出与资源属性检索 |
| 步骤 2 | 查看计划文件：审查已保存的执行计划 |
| 步骤 3 | JSON 机器可读输出：状态与计划的 JSON 格式解析 |

点击右侧箭头开始第一步。
