# terraform-docs 实战

通过本节实验，你将为一个 Terraform 模块自动生成文档，体验多种输出格式、README 注入和配置文件定制。

## 实验环境

工作目录中有一个 S3 静态网站托管模块（main.tf、variables.tf、outputs.tf），包含 7 个输入变量、4 个输出值和 4 个资源。

## 学习内容

| 内容 | 说明 |
|------|------|
| markdown / json / pretty | 多种输出格式 |
| tfvars hcl | 生成变量模板 |
| --output-file + inject | 将文档注入 README.md |
| .terraform-docs.yml | 配置文件定制 |
| content 模板 | 自定义文档结构 |

点击右侧箭头开始实验。
