# hcledit 实战练习

通过本节实验，你将在真实终端中掌握 hcledit 的核心命令，学会程序化地读取和修改 Terraform HCL 配置文件。

## 实验环境

hcledit 和 Terraform 已预装，工作目录中包含一组 Terraform 配置文件（main.tf、variables.tf、outputs.tf）。

## 学习内容

| 内容 | 说明 |
|------|------|
| attribute get / set | 读取和修改属性值 |
| attribute append / rm | 追加和删除属性 |
| block list / get / rm / new | 列出、查看、删除、新建块 |
| body get | 获取块体内容 |
| -f / -u 标志 | 文件模式与原地修改 |
| 管道组合 | 与其他命令配合使用 |

点击右侧箭头开始实验。
