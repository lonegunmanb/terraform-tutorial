# Terraform CLI 配置文件实战

通过本节实验，你将在真实终端中体验 Terraform CLI 配置文件的核心功能。

## 实验环境

Terraform 和 LocalStack 已预装并启动，但 terraform init 尚未执行——你将在实验中通过不同配置方式自行初始化。

## 学习内容

| 内容 | 说明 |
|------|------|
| .terraformrc 文件 | 创建并配置 CLI 配置文件 |
| plugin_cache_dir | 启用插件缓存，避免重复下载 |
| disable_checkpoint | 关闭版本检查网络请求 |
| provider_installation | 配置 filesystem_mirror 本地镜像 |
| TF_CLI_CONFIG_FILE | 用环境变量切换不同配置文件 |

点击右侧箭头开始实验。
